# Inkdown Feature Set

## Editor
- Full-screen writing mode as default (distraction-free, iA Writer-inspired)
- Floating draggable PDF preview bubble — expands on hover, shows live thumbnail
- Toggle to split-screen view (not the default — an option)
- Slash commands: /h1, /table, /code, /image for non-technical users
- Floating inline toolbar on text selection (bold, italic, link) — Notion-style
- Paste & Clean: detects AI-generated Markdown, strips artifacts automatically
- Auto-detect frontmatter and offer to use as document title/metadata
- CodeMirror 6 with custom Inkdown theme (dark + light)

## Export
- 5 curated PDF themes: Clean, Technical, Editorial, Minimal, Warm
- Theme picker sheet slides up before export
- Export animation — document 'prints' into PDF with satisfying micro-interaction
- Watermarked PDF on free tier. Clean PDF on paid tiers
- PDFShift/Browserless API for server-side generation with full CSS fidelity

## Mobile
- Writing mode only on mobile by default — no split screen
- Floating 'Preview PDF' button fixed at bottom — opens full-screen bottom sheet
- Export button always thumb-reachable (bottom, full width)
- Paste from Claude/ChatGPT directly into editor — primary mobile use case

## Subscriber Archive (Pro only)
- All past documents stored in Convex (raw .md + generated PDF)
- Real-time list view with search and sort
- Re-open, edit, and re-export any archived document
- Re-convert with a different theme without re-editing
